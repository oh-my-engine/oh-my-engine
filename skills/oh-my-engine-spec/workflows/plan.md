# Spec Plan Workflow

## Goal

Turn a proposal into a concrete implementation and verification plan.

## Execution Steps

### Step 1: Design refinement
```
1. Expand architecture and interfaces in design.md
2. Reconcile design choices with context/analysis.md and any imported prompt constraints
3. Refresh context/engine-memory.md from adopted learnings and generated skills
4. Document tradeoffs and rejected alternatives
5. Capture testing and rollout constraints
```

### Step 2: Task shaping
```
1. Break work into reviewable tasks
2. Include verification in the task list
3. Mark dependencies and follow-ups
```
