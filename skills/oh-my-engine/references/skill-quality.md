# Skill Quality Reference

OME skills must be executable and verifiable.

- Include Purpose, When to Use, Inputs, Process, Red Flags, Common Rationalizations, Verification, and Output Contract.
- Make triggers specific enough that an agent can decide when to use the skill.
- Include red flags that force reconsideration instead of guessing.
- Include anti-shortcut guidance for testing, scope, and verification.
- Reject skills that are only generic advice or cannot produce evidence.
