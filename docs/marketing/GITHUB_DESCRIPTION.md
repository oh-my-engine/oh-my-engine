## Product Hunt Description

**Tagline:**
```
Self-evolving workflow engine that learns and adapts
```

**Description:**
```
Oh My Engine provides a TypeScript-driven workflow layer for AI coding tools, with memory, specs, evolution candidates, and cross-tool rules.

🧠 Memory System
Remembers execution history, learnings, and user preferences across sessions.

🔄 Self-Evolution
Automatically identifies patterns and generates new skills when it detects:
- Repeated error fixes (≥3 times) → Auto-fix rules
- Code reuse (≥3 places) → Utility functions
- Successful patterns (≥95% success rate) → Best practices
- Operation sequences (≥5 times) → Quick commands

⚙️ Project Configuration
Per-project customization with `.ome/` directory for workflows, rules, and memory.

📋 Pre-built Workflows
- UI restoration from design files
- Bug analysis with context
- Component generation
- API integration

🌍 Multi-language Support
Documentation in English, Chinese, Japanese, Korean, Spanish, and French.

Perfect for developers who want their AI coding assistant to learn and improve over time!
```
